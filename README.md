# 🧠 Life Risk Algorithm

The algorithm uses:

1. Sum of all digits in birth date
2. Month value multiplied by `3`

---

## Example

```text
DOB: 21.07.2005

2 + 1 + 0 + 7 + 2 + 0 + 0 + 5 = 17

17 + (07 × 3) = 38
```

Result
Life Risk: 17 / 38
